## ğŸ› ï¸ Building from Source

GalaxyStrap is fully open-source and easy to build:

### Quick Build
```bash
# Clone the repository
git clone https://github.com/ethantheDeveloper220/vibetrap.git
cd vibetrap

# Build with .NET CLI
dotnet build GalaxyStrap.sln -c Release

# Or use the build script
.\build.bat
```

### Build Options
- **Standard Build:** `dotnet build GalaxyStrap.sln -c Release`  
- **Single-File:** `dotnet publish -c Release -r win-x64 --self-contained`  
- **Debug Build:** `dotnet build GalaxyStrap.sln -c Debug`

### Prerequisites
- .NET 8.0 SDK or Visual Studio 2022  
- Windows 7+ (x64)  

See [BUILD_INSTRUCTIONS.md](BUILD_INSTRUCTIONS.md) for detailed build documentation.
